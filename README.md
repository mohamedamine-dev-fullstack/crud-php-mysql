# CRUD PHP MySQL

##  Description

Mini projet CRUD pour gérer les étudiants.

##  Technologies

* PHP
* MySQL
* Bootstrap

##  Fonctionnalités

* Ajouter
* Modifier
* Supprimer
* Afficher

##  Installation

* Cloner le projet :
   git clone https://github.com/username/crud-php.git

* Déplacer le projet dans :
   C:\xampp\htdocs\crud_app

* Démarrer XAMPP :
   - Apache
   - MySQL

* Créer une base de données :
   crud_db

* Importer le fichier SQL dans phpMyAdmin

* Ouvrir le projet dans le navigateur :
   http://localhost/crud_app
